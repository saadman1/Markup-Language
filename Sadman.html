Introduction

Netflix is a subscription based video service offering movies, TV series, shows, etc. This project shows how video files in Netflix (or any similar service) can be stored and used. It includes data about movies, such as ratings, genre, etc. And it shows how users can filter the data acquiring the desired output.


Methodology

As it was required in this course, several technologies were used:

  • XML
  • XML Schema
  • HTML
  • XSL
  • XSLT

Project conctens

Using the aforementioned technologies, the following files have been created:

XML File
XML file containing elements and data about them
XML Schema File
XSD file describing the structure of XML
XSL Transforamtion File
Code for generation of HTML table
HTML File
Table with data obtained by XSLT

Elements

The root element is <netflix>. <netflix> contains 5 records of <video>.

Each <video> contains information about the movie, such as

  • <title>
  • <director>
  • <ratings>
  • <size>
  • <quality>
  • <popularity>
  • <recommendation>
  • <genre>
  • <year>
  • <restrictions>
  • <production>
  • etc.

Transformation

To use the data in XML, XSLT is used. With XSLT different kinds of output can be obtained.

5 video files available in Netflix are sorted by title name in this table. This table also contains other infromation about the video. If the video is recommended 80% and over, its cell is green. If the movie has popularity 60 or less, its cell is red.
